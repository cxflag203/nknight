Adding and working with nKnightControls

All nKnight controls have one very significant event: oError. If a user doesn't have access to a resource then nKnight will generate this event. For an example if a button is used and the user has access to the button, on clicking the button Button Click event shall be activated, otherwise it will activate the oError event.

1. nKnightbutton:

Access to this button will be validated when user click it, so you have to write the following,
private void nKnightButton1_Click(object sender, EventArgs e)
{
try
{
//Success, user have access to do this operation
}
catch (Exception ex)
{


}

}

If you want to get the error message then you have to get it like,
private void nKnightButton1_oError(nKnight.RBACControl.ControlEventArgs e)
{
MessageBox.Show(e.ErrorMessage);

}

2. nKnightCheckBox:

Access to this button will be validated when user click to check/ uncheck it, so the coding is same as nKnightButton.

3. nKnightCheckListBox:

Access to this listbox will be validated when user will check/ uncheck any item in the listview. So the coding will be same but event will be

private void nKnightCheckListBox1_ItemCheck(object sender, ItemCheckEventArgs e).

4. nKnightComboBox:

Access to this combobox will be validated when user change any selection in the combobox. So the event will be,

private void nKnightCombo1_SelectionChangeCommitted(object sender, EventArgs e)

5. nKnightDateTimePicker:

Access to this control will be validated when user leave the control. So the event is,

private void nKnightDateTimePicker1_Validating(object sender, CancelEventArgs e)

6. nKnightGridView:

Access to this control will be validated when user will try to change something in the cell. So the event will be,

private void dataGridView1_CellBeginEdit(object sender, DataGridViewCellCancelEventArgs e)

7. nKnightLinkLabel:

Access to this control will be validated, when user click it. So the event will be,

private void nKnightLinkLabel1_LinkClicked(object sender, LinkLabelLinkClickedEventArgs e)

8. nKnightMaskedTextBox:

Access to this control will be validated when user try to write something. So the event will be,

private void nKnightMaskedTextBox1_KeyPress(object sender, KeyPressEventArgs e)

9. nKnightRadioButton:

Access to this control will be validated when user try to check/ uncehck it. So the event is,

private void nKnightRadioButton1_KeyPress(object sender, KeyPressEventArgs e)

10. nKnightTextBox:

Access to this control will be validated when user will try to write something. So the event is,

private void nKnightTextBox1_KeyPress(object sender, KeyPressEventArgs e)
